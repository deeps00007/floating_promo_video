# Changelog for floating_promo_video

## [0.1.0] - 2023-10-01
- Initial release of the `floating_promo_video` package.
- Added `FloatingPromoVideo` widget for displaying floating promotional videos.
- Implemented draggable behavior for the floating video.
- Created `ExpandableVideoCard` widget for expanding and collapsing video cards.
- Developed `StoryPlaylist` widget for managing a sequence of videos.
- Introduced `ProgressBar` widget to show video playback progress.
- Added `FloatingVideoController` for managing video playback state.
- Included `PlaylistController` for handling video playlists.
- Defined `PromoVideo` and `PlaylistItem` models for video and playlist data.
- Implemented utility functions for draggable behavior in `drag_behavior.dart`.
