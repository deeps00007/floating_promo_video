# Changelog for floating_promo_video

## [1.0.0] - 2026-03-17
- Initial Stable Release of the `floating_promo_video` package 🚀
- Implemented `FloatingPromotionVideo` for dragging and playing videos on screen.
- Added drop-in `FloatingPromoVideoScaffold` and `FloatingPromoVideoOverlay` wrappers.
- Integrated `tokenApiUrl` support for securely fetching Instagram Long-Lived Access Tokens.
- Built-in video fallback mechanism for CDN MP4 URLs.
- Seamless "story-style" progress bar playlist for watching multiple reels.
- Draggable physics that snap accurately to screen edges.
- Animated Mute and Close widgets for easy session management.
